# Tools

- 2 Cat5e ethernet Cable (5 metres recommended)
- Wire Stripper/Knife
- Scissors

# STEPS
- Take the first cat5e cable and expose the wires inside (with knife/stripper)
- For second cable, ****cut off one end and remove all wires, EXCEPT : WHITE-GREEN AND GREEN WIRE**
- Connect WHITE-GREEN and GREEN from second cable --> to WHITE-GREEN and GREEN of exposed cable
