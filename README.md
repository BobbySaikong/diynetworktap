# Tools

- Cat5e Cable (5 metres recommended)
- 3 rj45 Cat5/Cat5e connectors
- 1x Single Port Cat5e Wall Jack
- Wire Stripper
- rj45 crimp tool
- Scissors
- Electrical Tape

# STEP 1
- Cut a 5metres cat5e cable into a 2 metre with Scissors.
- Install RJ45 connectors on both ends with the crimp tool.
